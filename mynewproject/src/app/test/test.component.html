Component Interaction

To display message from App component to Test Component:

  • In AppComponent.ts create a new property public name="Namrata"
  • Then in App.Component.html, include it in the opening tag of the selecter bind the name property
  • Now send the data to the TestComponent.ts. Declare the same property that the App Component is sending using Input decorator, @Input() public parentData. You can even create an Alias for eg; @Input('parentData') public name; Also import it from @angular/Core - import {Input} from '@angular/core'
  • Now bind it into the template,

    {{"Hello " + parentData}}

To display message from Test Component to App Component. In Child component you have parent component selector but in parent component you do not have child component selecter so you have to send the message using an event.

  • In test.component.ts create a property - @Output() public childEvent = new EventEmitter(); Import the EventEmitter & Output class import{Output, EventEmitter} from '@angular/core';
  • On click lets call a fireEvent method and define it in export class
  • Then go to app.component.html and bind custom event and assign it to the property called message the event variable - (childEvent)="message&event". The $event variable is going to refer "Hey CodeEvolution"
  • Declare the property message - public message = ""
  • Display it in the template {{message}}
